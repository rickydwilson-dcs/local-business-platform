# @platform/core-components

## 1.1.0

### Minor Changes

- 720a782: Add Hero component variants (V1, V2, V3)

  This release introduces three hero component variants to support different business types and design preferences:

  **HeroV1 - Classic Centered Hero**
  - Simple, centered layout
  - Title, subtitle, and single CTA
  - Best for traditional businesses and professional services

  **HeroV2 - Split Layout Hero**
  - Two-column layout with content and visual
  - Support for primary and secondary CTAs
  - Optional features list
  - Best for modern businesses, SaaS, tech companies

  **HeroV3 - Full-Screen Hero**
  - Immersive full-screen layout
  - Support for video or image backgrounds
  - Customizable overlay opacity
  - Scroll indicator animation
  - Best for creative agencies, portfolios, hospitality

  All variants are fully typed with TypeScript and support customizable props for maximum flexibility.
